---
title: keysIn
definition: 
description: This method returns all the own and inherited enumerable property names of an object.
---


#### Arguments


```bash
{object} object - The object to query.
```


#### Returns


```bash
{Array} - Returns the array of property names.
```


#### Example


```ts
keysIn({a: 1, b: 2})  // => ['a', 'b']
```
