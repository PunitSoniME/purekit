---
title: head
definition: head(array)
description: Returns the first element of an array, or undefined if the array is empty.
---


#### Arguments


```bash
{Array} array - The array to query.
```


#### Returns


```bash
{*}: Returns the first element of array.
```


#### Example


```ts
head([1, 2, 3]); // => 1
head([]); // => undefined
```
