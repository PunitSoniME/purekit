---
title: last
definition: last(array)
description: Returns the last element of an array.
---

#### Arguments

```bash
{Array} array - The array to query.
```

#### Returns

```bash
{*} - Returns the last element of the array.
```

#### Example

```ts
_.last([1, 2, 3])
// => 3

_.last([])
// => undefined
```
