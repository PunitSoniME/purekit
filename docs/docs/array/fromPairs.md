---
title: fromPairs
definition: 
description: Creates an object from an array of key-value pairs.
---


#### Arguments


```bash
{T[][]} array - The array of key-value pairs to convert to an object.
```


#### Returns


```bash

```


#### Example


```ts
fromPairs([['a', 1], ['b', 2], ['c', 3]]); // {a: 1, b: 2, c: 3}
```
