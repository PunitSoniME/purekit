---
title: 'compact'
description: 'compact method'
---

```ts
compact(array)
```

#### Arguments

```bash
- array (Array): The array to compact.
```

#### Returns

```bash
- (Array): Returns the new array of filtered values.
```

#### Example

```ts
import { compact } from 'alt-lodash';

compact([0, 1, false, 2, '', 3]);
// => [1, 2, 3]
```
