---
title: degreesToRads
definition: degreesToRads(deg)
description: Converts degrees to radians.
---


#### Arguments


```bash
{number} deg - The value in degrees.
```


#### Returns


```bash
{number} - The value converted to radians.
```


#### Example


```ts
_.degreesToRads(90);
// => 1.5707963267948966

_.degreesToRads(180);
// => 3.141592653589793
```
