description: Returns the size of the given collection.
---


#### Arguments


```bash
{Array|Object|string} collection - The collection to determine the size of.
```


#### Returns


```bash

```


#### Example


```ts
size([1, 2, 3]); // returns 3size({a: 1, b: 2, c: 3}); // returns 3size('hello'); // returns 5
```
