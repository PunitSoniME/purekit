---
title: toPairs
definition: 
description: Creates an array of key-value pairs for the given object.
---


#### Arguments


```bash
{object} object - The object to convert to pairs.
```


#### Returns


```bash

```


#### Example


```ts
const obj = { a: 1, b: 2, c: 3 };const result = toPairs(obj);console.log(result); // [['a', 1], ['b', 2], ['c', 3]]
```
