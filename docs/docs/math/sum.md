---
title: sum
definition: sum(array)
description: Returns the sum of all the numbers in the array.
---


#### Arguments


```bash
{number[]} numbers - The array of numbers to sum.
```


#### Returns


```bash
{number} - The sum of all the numbers in the array.
```


#### Example


```ts
sum([1, 2, 3, 4, 5]);
// => 15

sum([-1, -2, 0, 1]);
// => -2
```
