---
title: pad
definition: 
description: Pads a string with a specified character to a specified length.
---


#### Arguments


```bash
{string} str - The string to pad.
{number} length - The length to pad the string to.
```


#### Returns


```bash

```


#### Example


```ts
pad('hello', 10); // '  hello   'pad('hello', 10, '-'); // '--hello---'
```
