---
title: isUndefined
definition: 
description: Checks if a value is undefined.
---


#### Arguments


```bash
{any} value - The value to check.
```


#### Returns


```bash

```


#### Example


```ts
isUndefined(undefined); // trueisUndefined(null); // falseisUndefined(0); // falseisUndefined(''); // falseisUndefined(false); // false
```
